# 異世界にネットがなかったので、魔法で作ります。

## About

本作は、  
魔法が存在する一方で、情報インフラが存在しない異世界に転生した主人公が、  
魔法を用いて「ネットワーク的な仕組み」を構築していく物語です。

本文は小説家になろうに投稿しつつ、  
本リポジトリでは本文・世界観設定・メタ情報を管理します。

小説を「ソフトウェアプロジェクト」のように育てることを目的としています。

---

## Repository Structure

- `novel/`  
  小説本文（プロローグ・各 bit）

- `world/`  
  世界観・魔法・魔素・応答体などの設定資料

- `characters/`  
  登場人物の設定

- `meta/`  
  執筆方針、タイムライン、アイデアメモなど

- `assets/`  
  ロゴ等の素材
